## 🗝 Installation and usage

---

Simple exercice, simple installation.

### 🚙 Running the project

1️. Clone this repository

`git clone https://github.com/manon-boiteau/prog-tv-LeReacteur.git`

`cd prog-tv-LeReacteur`

2️. Install packages

`npm install`  
or  
`yarn`

3️. When installation is complete:

`yarn start`
